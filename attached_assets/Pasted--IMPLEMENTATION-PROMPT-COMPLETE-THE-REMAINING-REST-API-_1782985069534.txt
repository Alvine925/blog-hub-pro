# TASK 3 — VERIFY WORKSPACE ISOLATION

Audit every database query used by REST endpoints.

Every query must automatically filter by the resolved workspace.

Example

Incorrect

```sql

FROM blogs;
```

Correct

```sql

FROM blogs
```

The client must never send

* workspace_id

The API Key must resolve the workspace automatically.
