# Task: Implement the Lunar CMS Dashboard & Workspace UI

## Objective

Implement a modern, premium SaaS dashboard inspired by the provided design mockups.

The UI should feel like a blend of:

* Linear
* Vercel
* Supabase
* Stripe Dashboard
* Notion
* GitHub

The experience should be minimal, extremely clean, responsive, and fast.

This implementation should prioritize component reusability, scalability, and maintainability.

The project should use the existing design system wherever possible while introducing new reusable components as needed.

---

# General Design Language

The application should communicate:

* Professional
* Premium
* Fast
* Modern
* Developer Friendly

Avoid clutter.

Avoid excessive colors.

Use whitespace generously.

Rounded corners.

Subtle shadows.

Final Goal

The completed interface should feel like a premium SaaS product rather than a typical admin dashboard. Users should immediately understand the separation between the Global Dashboard (where they manage workspaces) and the Workspace Dashboard (where they manage content). The UI should be highly polished, responsive, scalable, and built entirely with reusable components so future modules can be added without redesigning the application.
