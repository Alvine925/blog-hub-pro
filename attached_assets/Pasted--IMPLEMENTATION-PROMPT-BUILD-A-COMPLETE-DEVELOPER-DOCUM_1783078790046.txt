# IMPLEMENTATION PROMPT

# BUILD A COMPLETE DEVELOPER DOCUMENTATION PORTAL FOR LUNAR CMS

## Objective

Transform the existing Documentation section into a **world-class Developer Documentation Portal** similar in quality to Stripe, Supabase, Vercel, Clerk, and OpenAI.

This is **not** a simple documentation page.

It is a complete documentation application with its own navigation, layout, search, examples, AI integration, interactive API explorer, implementation guides, code generators, and developer resources.

The purpose is to enable a developer to integrate Lunar CMS into any website or application without requiring external assistance.

---

# IMPORTANT

Before implementing anything, inspect the existing project.

The following systems already exist and **must be reused** where appropriate.

* Authentication
* Workspaces
* API Keys
* REST API
* API Explorer
* AI Documentation Assistant
* Code Example Generator
* Developer Dashboard
* Existing Documentation
* Existing UI Components
* Existing Design System
* Existing Theme
* Existing Routing
* Existing Database
* Existing Search

Do **not** duplicate functionality.

Extend the existing architecture.

